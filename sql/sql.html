SQL stands for Structured Query Language, an ANSI (American National Standards Institute) standard language used to manage and communicate with databases, i.e., database creation, deletion, update rows by fetching rows, modifying rows, many more. It has many versions like T-SQL by Microsoft, PSQL by Interbase/ Firebird, many more. Therefore, the most common relational database management systems which use SQL are MS SQL Server, Sybase, DB2, Oracle, Ingres, Access. Although most of these database systems use SQL, most have other additional proprietary extensions made particularly for their system. Hence, we will be only focusing on the ANSI/ ISO standard certified SQL language.
